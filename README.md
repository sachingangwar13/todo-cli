
# TODO CLI

A command-line interface for managing your todo tasks efficiently.

## Installation

```bash
git clone https://github.com/sachingangwar13/todo-cli.git
cd todo-cli
npm install
npm link
```

## Usage

```bash
# Add a new todo
todo add "Your task here"

# List all todos
todo list

# Mark todo as complete
todo complete <id>

# Delete a todo
todo delete <id>
```

## Requirements

- Node.js 14+
- npm or yarn

## Getting Started

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to launch the CLI
4. Follow the prompts to manage your tasks
5. Run `npm link` to  use the commands globally.
